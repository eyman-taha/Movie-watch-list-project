# CineWatch - UI/UX Design Package

## Overview

This folder contains all UI/UX design documentation and assets for the CineWatch Flutter application.

## Contents

```
UI-UX-Design/
├── wireframes/          # Low-fidelity wireframes (ASCII/SVG)
├── mockups/            # High-fidelity design mockups
├── docs/               # Design documentation
└── README.md           # This file
```

## Design Tools Used

- **Wireframing**: Hand-drawn sketches + ASCII diagrams
- **Mockups**: Flutter widgets + design specifications
- **Documentation**: Markdown + ASCII art diagrams

## Screens Included

1. **Splash Screen** - App branding and loading
2. **Login/Register** - Authentication flow
3. **Home Screen** - Movie discovery with categories
4. **Movie Details** - Full movie information
5. **Search Screen** - Movie search with filters
6. **Watchlist** - Personal movie tracking
7. **Profile** - User settings and statistics

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary (Red) | #E50914 | Buttons, accents |
| Secondary (Gold) | #FFD700 | Ratings, highlights |
| Background Dark | #0D0D0D | Dark mode background |
| Surface Dark | #1A1A1A | Cards, containers |

| Style | Font | Size | Weight |
|-------|------|------|--------|
| Display | System | 32px | Bold |
| Title | System | 24px | Bold |
| Headline | System | 20px | SemiBold |
| Body | System | 16px | Regular |
| Caption | System | 12px | Regular |

## Design Principles

1. **Minimalist**: Clean interfaces with focused content
2. **Dark-First**: Dark mode as primary theme
3. **Card-Based**: Content organized in rounded cards
4. **Smooth Animations**: Hero transitions and micro-interactions
5. **Consistent Spacing**: 8px grid system
